# 小游戏系统设计 - UI Context 隔离机制

**文档版本**: v1.0  
**生成时间**: 2026-05-20  
**设计目标**: 解决小游戏模块与主游戏UI系统的冲突，实现完全隔离的运行环境

---

## 1. 设计背景

### 1.1 当前问题

小游戏模块在集成到主游戏系统时，存在以下核心问题：

| 问题 | 原因 | 影响 |
|------|------|------|
| 健身房指示条不动 | `deltaTime` 单位不一致（秒 vs 毫秒） | 小游戏完全不可用 |
| 工作模态框不关闭 | 小游戏启动时未清除主游戏UI | 界面遮挡 |
| 位置判定错误 | 使用相对坐标与绝对坐标比较 | 判定逻辑失效 |
| 时间计算不暂停 | 使用 `Date.now()` 而非统一时间源 | 体验混乱 |

### 1.2 设计目标

```
┌─────────────────────────────────────────────────────────────┐
│                        设计目标                              │
├─────────────────────────────────────────────────────────────┤
│  1. 小游戏拥有完全独立的UI生命周期                            │
│  2. 小游戏运行期间，主游戏状态暂停但保持完整                  │
│  3. 统一的触摸事件路由机制                                   │
│  4. 统一的时间管理系统                                       │
│  5. 清晰的进入/退出流程                                     │
└─────────────────────────────────────────────────────────────┘
```

### 3.1 MiniGameContext 类

```javascript
/**
 * 小游戏上下文管理器
 * 负责管理小游戏的生命周期、状态、事件路由
 */
class MiniGameContext {
    
    // ==================== 状态定义 ====================
    static State = {
        IDLE: 'idle',           // 未激活
        ACTIVE: 'active',       // 运行中
        PAUSED: 'paused',       // 暂停
        COMPLETED: 'completed'  // 已完成
    }
    
    // ==================== 构造函数 ====================
    constructor(game) {
        this.game = game
        this.state = MiniGameContext.State.IDLE
        this.activeMiniGame = null
        this.savedUIState = null
        this.onCompleteCallback = null
    }
    
    // ==================== 生命周期 ====================
    
    /**
     * 进入小游戏模式
     * @param {string} miniGameType - 小游戏类型标识
     * @param {object} config - 小游戏配置
     * @param {function} onComplete - 完成回调
     */
    enter(miniGameType, config, onComplete) {
        // 1. 保存当前游戏状态
        this._saveGameState()
        
        // 2. 清除主游戏UI
        this._clearMainUI()
        
        // 3. 创建小游戏实例
        this._createMiniGame(miniGameType, config)
        
        // 4. 更新状态
        this.state = MiniGameContext.State.ACTIVE
        this.onCompleteCallback = onComplete
        
        // 5. 暂停主游戏逻辑更新
        this.game.pauseMainUpdate = true
        
        console.log(`[MiniGameContext] 进入小游戏: ${miniGameType}`)
    }
    
    /**
     * 退出小游戏模式
     * @param {number} successRate - 成功率
     */
    exit(successRate = 0) {
        // 1. 标记完成
        this.state = MiniGameContext.State.COMPLETED
        
        // 2. 清理小游戏
        this._destroyMiniGame()
        
        // 3. 恢复主游戏UI
        this._restoreMainUI()
        
        // 4. 恢复主游戏逻辑
        this.game.pauseMainUpdate = false
        
        // 5. 触发完成回调
        if (this.onCompleteCallback) {
            this.onCompleteCallback(successRate)
        }
        
        // 6. 重置状态
        this.state = MiniGameContext.State.IDLE
        
        console.log(`[MiniGameContext] 退出小游戏, 成功率: ${successRate}`)
    }
    
    // ==================== 事件处理 ====================
    
    /**
     * 处理触摸开始事件
     */
    handleTouchStart(x, y) {
        if (this.state !== MiniGameContext.State.ACTIVE) {
            return false
        }
        
        if (this.activeMiniGame) {
            return this.activeMiniGame.handleTouch(x, y)
        }
        
        return false
    }
    
    /**
     * 处理触摸移动事件
     */
    handleTouchMove(x, y) {
        if (this.state !== MiniGameContext.State.ACTIVE) {
            return
        }
        
        if (this.activeMiniGame && this.activeMiniGame.handleTouchMove) {
            this.activeMiniGame.handleTouchMove(x, y)
        }
    }
    
    /**
     * 处理触摸结束事件
     */
    handleTouchEnd(x, y) {
        if (this.state !== MiniGameContext.State.ACTIVE) {
            return false
        }
        
        if (this.activeMiniGame && this.activeMiniGame.handleTouchEnd) {
            return this.activeMiniGame.handleTouchEnd(x, y)
        }
        
        return false
    }
    
    // =================───┐
    // 私有方法           │
    // ====================
    
    _saveGameState() {
        this.savedUIState = {
            modals: [...this.game.uiManager.modals],
            buttons: [...this.game.uiManager.buttons]
        }
    }
    
    _clearMainUI() {
        this.game.uiManager.closeAllModals()
        this.game.uiManager.clearAll()
    }
    
    _restoreMainUI() {
        if (this.savedUIState) {
            // 可选择是否恢复之前状态
            // this.game.uiManager.modals = this.savedUIState.modals
            this.savedUIState = null
        }
    }
    
    _createMiniGame(type, config) {
        const MiniGameFactory = {
            'work': WorkMiniGame,
            'gym': GymMiniGame,
            // 未来可扩展更多小游戏
        }
        
        const MiniGameClass = MiniGameFactory[type]
        if (!MiniGameClass) {
            console.error(`[MiniGameContext] 未知小游戏类型: ${type}`)
            return
        }
        
        // 创建实例，传入时间提供者
        this.activeMiniGame = new MiniGameClass(
            this.game,
            this.game.timeProvider,
            (successRate) => this._onMiniGameComplete(successRate)
        )
        
        this.activeMiniGame.start(config)
    }
    
    _destroyMiniGame() {
        if (this.activeMiniGame) {
            this.activeMiniGame.destroy()
            this.activeMiniGame = null
        }
    }
    
    _onMiniGameComplete(successRate) {
        this.exit(successRate)
    }
}
```

### 3.2 TimeProvider 类

```javascript
/**
 * 统一时间提供者
 * 解决不同模块对时间单位理解不一致的问题
 */
class TimeProvider {
    
    constructor() {
        this._lastTimestamp = 0
        this._deltaTime = 0        // 秒为单位
        this._elapsed = 0          // 游戏运行总时间（秒）
        this._paused = false
        this._timeScale = 1.0
    }
    
    /**
     * 更新（每帧调用）
     * @param {number} timestamp - requestAnimationFrame 提供的时间戳
     */
    update(timestamp) {
        if (this._lastTimestamp === 0) {
            this._lastTimestamp = timestamp
            return
        }
        
        if (this._paused) {
            this._deltaTime = 0
            return
        }
        
        const rawDelta = (timestamp - this._lastTimestamp) / 1000  // 转换为秒
        this._deltaTime = rawDelta * this._timeScale
        this._elapsed += this._deltaTime
        this._lastTimestamp = timestamp
    }
    
    // ==================== 属性 ====================
    
    /** 帧间隔时间（秒） */
    get deltaTime() {
        return this._deltaTime
    }
    
    /** 帧间隔时间（毫秒） */
    get deltaTimeMs() {
        return this._deltaTime * 1000
    }
    
    /** 游戏运行总时间（秒） */
    get elapsed() {
        return this._elapsed
    }
    
    /** 标准化deltaTime（假设60fps为1.0） */
    get normalizedDelta() {
        return this._deltaTime / (1 / 60)
    }
    
    // ==================== 控制方法 ====================
    
    pause() {
        this._paused = true
    }
    
    resume() {
        this._paused = false
        this._lastTimestamp = 0  // 重置以避免时间跳跃
    }
    
    setTimeScale(scale) {
        this._timeScale = Math.max(0, Math.min(2, scale))
    }
    
    reset() {
        this._elapsed = 0
        this._lastTimestamp = 0
        this._deltaTime = 0
    }
}
```

```javascript
 * 所有小游戏都应继承此类

class MiniGame {
    
    constructor(game, timeProvider, onComplete) {
        this.game = game
        this.time = timeProvider  // 使用统一时间源
        this.onComplete = onComplete
        
        this.active = false
        this.phase = 'idle'
    }
    
    /**
     * 启动小游戏
     * @param {object} config - 配置参数
     */
    start(config) {
        this.active = true
        this.phase = 'playing'
        this._initState(config)
        console.log(`[${this.constructor.name}] 小游戏启动`)
    }
    
    /**
     * 更新（每帧调用）
     */
    update() {
        if (!this.active) return
        
        // 使用 this.time.deltaTime 而非假设单位
        this._updateLogic(this.time.deltaTime)
    }
    
    /**
     * 渲染
     */
    render(renderer) {
        if (!this.active) return
        this._renderUI(renderer)
    }
    
    /**
     * 处理触摸
     */
    handleTouch(x, y) {
        if (!this.active || this.phase !== 'playing') return false
        return this._handleTouch(x, y)
    }
    
    /**
     * 完成小游戏
     */
    complete(successRate) {
        this.active = false
        this.phase = 'completed'
        
        if (this.onComplete) {
            this.onComplete(successRate)
        }
    }
    
    /**
     * 销毁
     */
    destroy() {
        this.active = false
        this.phase = 'idle'
    }
    
    // ==================== 抽象方法 ====================
    
    _initState(config) {

    _updateLogic(deltaTime) {

    }

    }
```

```javascript
class WorkMiniGame extends MiniGame {
    
    constructor(game, timeProvider, onComplete) {
        super(game, timeProvider, onComplete)
        
        // 基础配置
        this.totalTargets = 8
        this.requiredHits = 6
        this.timeLimit = 5000  // 毫秒
        this.targetLifetime = 1500
        
        // 状态
        this.targets = []
        this.hitCount = 0
        this.startTime = 0
        this.gameElapsed = 0  // 使用游戏时间
    }
    
    _initState(config) {
        this.targets = []
        this.hitCount = 0
        this.startTime = this.time.elapsed
        this.gameElapsed = 0
        this._spawnTargets()
    }
    
    _updateLogic(deltaTime) {
        if (this.phase !== 'playing') return
        
        // 关键修复：使用 time.elapsed 而不是 Date.now()
        this.gameElapsed = this.time.elapsed
        
        const currentMs = this.gameElapsed * 1000  // 转为毫秒
        
        // 检查游戏时间
        if (currentMs >= this.timeLimit) {
            this._finish()
            return
        }
        
        // 更新目标状态
        this._updateTargets(currentMs)
    }
    
    _spawnTargets() {
        const w = this.game.renderer.width
        const h = this.game.renderer.height
        const spawnInterval = (this.timeLimit / 1000) / this.totalTargets  // 秒
        
        for (let i = 0; i < this.totalTargets; i++) {
            this.targets.push({
                x: 60 + Math.random() * (w - 120),
                y: 140 + Math.random() * (h - 280),
                spawnTime: this.startTime + i * spawnInterval,  // 使用游戏时间
                radius: 20,
                hit: false,
                expired: false
            })
        }
    }
    
    _updateTargets(currentMs) {
        for (const target of this.targets) {
            if (target.hit || target.expired) continue
            
            const age = currentMs - (target.spawnTime * 1000)  // 转为毫秒
            
            if (age >= this.targetLifetime) {
                target.expired = true
            } else if (age >= this.targetLifetime * 0.7) {
                target.opacity = 1.0 - (age - this.targetLifetime * 0.7) / (this.targetLifetime * 0.3)
            }
        }
    }
    
    _handleTouch(x, y) {
        if (this.phase !== 'playing') return false
        
        const currentMs = this.gameElapsed * 1000
        
        for (const target of this.targets) {
            if (target.hit || target.expired) continue
            
            const age = currentMs - (target.spawnTime * 1000)
            if (age < 0) continue  // 未到生成时间
            
            const dx = x - target.x
            const dy = y - target.y
            const distance = Math.sqrt(dx * dx + dy * dy)
            
            if (distance <= 32) {  // touchRadius
                target.hit = true
                this.hitCount++
                this._spawnParticles(target.x, target.y)
                return true
            }
        }
        
        return false
    }
}
```

```javascript
class Game {
    constructor() {
        // ... 现有初始化 ...
        
        // 新增组件
        this.timeProvider = new TimeProvider()
        this.miniGameContext = new MiniGameContext(this)
    }
    
    async initGame() {

        // 确保 TimeProvider 在其他组件之前初始化
        this.timeProvider = new TimeProvider()
        this.miniGameContext = new MiniGameContext(this)
    }
    
    loop() {
        const now = Date.now()
        
        // 关键：更新时间提供者
        this.timeProvider.update(now)
        
        // 只有未暂停时才更新
        if (!this.pauseMainUpdate) {
            this.update()
        }
        
        this.render()
        
        // 无论是否暂停，都更新小游戏
        if (this.miniGameContext.state === MiniGameContext.State.ACTIVE) {
            this.miniGameContext.activeMiniGame.update()
        }
        
        requestAnimationFrame((ts) => this.loop(ts))
    }
    
    // 新增方法：启动小游戏
    startMiniGame(type, config, onComplete) {
        this.miniGameContext.enter(type, config, onComplete)
    }
}
```

```javascript
initEvents() {
    wx.onTouchStart((e) => {
        const touch = e.touches[0]
        const x = touch.clientX
        const y = touch.clientY
        
        // 优先处理调试面板

            return

            return
        }
        
        this.sceneManager.handleTouchMove(x, y)
    })
    
    wx.onTouchEnd((e) => {
        const touch = e.changedTouches[0]
        const x = touch.clientX

            return
        }

        if (this.uiManager.modals.length > 0) {
            return
        }
        
        // 小游戏处理
        if (this.miniGameContext.state === MiniGameContext.State.ACTIVE) {
            this.miniGameContext.handleTouchEnd(x, y)
            return
        }
        
        this.sceneManager.handleTouchEnd(x, y)
    })
}
```

```javascript
                })
                return
            }
        }
        
        // 关键：使用统一的启动方式
        this.game.startMiniGame('work', {
            salary: salary,
            isOvertime: isOvertime
        }, (successRate) => {
            this.doWork(isOvertime, Math.floor(salary * successRate), successRate, onComplete)
        })
    }
}
```

### 5.2 渲染逻辑

```javascript
// Game.render()
render() {
    this.ctx.clearRect(0, 0, this.width, this.height)
    
    // L1: 场景背景（小游戏运行时仍然渲染，但可能半透明）
    if (this.miniGameContext.state !== MiniGameContext.State.ACTIVE) {
        this.sceneManager.render(this.renderer)
    }
    
    // L2: 主游戏UI（小游戏运行时不渲染）
    if (this.miniGameContext.state !== MiniGameContext.State.ACTIVE) {
        this.uiManager.render(this.renderer)
    }
    
    // L3: 小游戏UI
    if (this.miniGameContext.state === MiniGameContext.State.ACTIVE) {
        this.miniGameContext.render(this.renderer)
    }
    
    // L5: 调试面板
    if (this.debugPanel && this.debugPanel.isVisible) {
        this.debugPanel.render()
    }
}

// MiniGameContext.render()
render(renderer) {
    // 小游戏拥有完全独立的渲染空间
    if (this.activeMiniGame) {
        this.activeMiniGame.render(renderer)
    }
}
```

## 8. 优势总结

| 优势 | 说明 |
|------|------|
| **完全隔离** | 小游戏拥有独立的UI层和事件处理，不受主游戏干扰 |
| **统一时间** | 所有时间相关逻辑使用 TimeProvider，避免单位混乱 |
| **清晰生命周期** | enter() / exit() 明确管理进入和退出 |
| **易于扩展** | 新增小游戏只需实现 MiniGame 基类 |
| **状态可控** | 统一的游戏状态管理 |
| **渲染层级清晰** | 明确的渲染顺序，避免UI重叠 |
