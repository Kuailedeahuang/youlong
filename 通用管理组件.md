# 通用管理组件

> 本文档记录项目中已实现统一管理、可直接复用的组件信息。

## 一、核心系统 (Core)

### 1. SceneManager — 场景管理器

| 项目 | 说明 |
|:---|:---|
| **路径** | [js/core/sceneManager.js](js/core/sceneManager.js) |
| **导出** | `default class` |
| **依赖** | 各 Scene 子类 |
| **用途** | 统一管理所有游戏场景的注册、切换和生命周期。内置防抖机制防止重复切换。 |

**关键 API：**

```javascript
// 直接切换场景
sceneManager.switchTo('home')

// 带参数切换场景
sceneManager.switchToWithParams('sceneWithBackground', { sceneName: 'home' })

// 按地点名称跳转
sceneManager.goToLocation('bank')

// 获取场景实例
sceneManager.getScene('home')

// 生命周期（帧循环内调用）
sceneManager.update(deltaTime)
sceneManager.render(renderer)
```

**已注册场景：** `login` / `splash` / `home` / `market` / `house` / `sceneWithBackground` / `settings` / `map` / `computer`

### 2. GameState — 游戏状态中心

| 项目 | 说明 |
|:---|:---|
| **路径** | [js/core/gameState.js](js/core/gameState.js) |
| **导出** | `default class` |
| **依赖** | `GAME_CONFIG`, `ItemData` |
| **用途** | 数据核心。管理所有玩家属性、仓库、每日结算、价格更新、报纸事件、延迟动画队列，支持云数据库 + 本地存储双持久化。 |

**关键 API：**

```javascript
// 键值读写
gameState.get('money')
gameState.set('currentScene', 'home')

// 属性批量操作
gameState.addMoney(100)
gameState.addEnergy(-1)
gameState.addHealth(5)

// 进入下一天（核心逻辑）
gameState.nextDay()

// 延迟动画队列
gameState.addDelayedAnimation('increase', 100, 'money', '金币', '#f39c12')
const anims = gameState.getAndClearDelayedAnimations()

// 商品价格
gameState.updateItemPrices()
gameState.generateNewspaperEvents()

// 仓库
gameState.addToWarehouse('item_1', 10, 500)
gameState.removeFromWarehouse('item_1', 5)
```

### 3. Renderer — 渲染引擎

| 项目 | 说明 |
|:---|:---|
| **路径** | [js/core/renderer.js](js/core/renderer.js) |
| **导出** | `default class` |
| **依赖** | `IconManager` |
| **用途** | Canvas 2D 渲染封装层。提供吉卜力风格的绘图原语（面板、按钮、文字、进度条、场景元素）。 |

**关键 API：**

```javascript
const renderer = new Renderer(ctx, width, height)

// 基础绘制
renderer.clear('#16213e')
renderer.drawRect(x, y, w, h, color, radius)
renderer.drawText(text, x, y, color, fontSize, align)
renderer.drawButton(x, y, w, h, text, bgColor, textColor, fontSize)
renderer.drawProgressBar(x, y, w, h, progress, bgColor, fillColor)
renderer.measureText(text, fontSize)

// 吉卜力专属
renderer.drawGhibliPanel(x, y, w, h, bgColor)
renderer.drawGhibliButton(x, y, w, h, text, bgColor, textColor, fontSize)

// 复合渲染
renderer.renderStatsPanel(game, state)      // 底部属性面板
renderer.renderCenterButton(game, x, y, size) // 中央地图按钮
renderer.renderStatItem(x, y, statType, value, max, color)
```

### 4. UIManager — UI 管理器

| 项目 | 说明 |
|:---|:---|
| **路径** | [js/core/uiManager.js](js/core/uiManager.js) |
| **导出** | `default class` |
| **依赖** | 无 |

```javascript
    content: '内容文字',

    itemName: '数量', price: 1, quantity: 10, maxQuantity: 100,
    tradeType: 'deposit',
    onConfirm: (qty) => { }
})

uiManager.closeModal()
uiManager.closeAllModals()
uiManager.clearAll()
```

### 5. IconManager — 图标管理器

| 项目 | 说明 |
|:---|:---|
| **路径** | [js/components/IconManager.js](js/components/IconManager.js) |
| **导出** | 单例 `iconManager` |
| **依赖** | `SVGIcons` |
| **用途** | 统一管理所有 SVG 图标的绘制入口。通过名称字符串分发到对应的 SVG 绘制方法，自动应用主题轮廓线和投影。 |

**关键 API：**

```javascript
// 绘制单个图标
iconManager.draw(ctx, 'health', x, y, { size: 'md', color: '#7CB87C' })

// 批量绘制
iconManager.drawBatch(ctx, [
    { name: 'health', x: 50, y: 50, options: { size: 'lg' } },
    { name: 'energy', x: 100, y: 50, options: { size: 'lg' } }
])

// 注册新图标
iconManager.register('myIcon', drawFunction, { label: '新图标', color: '#xxx' })

// 获取已注册图标名
iconManager.getIconNames()
```

**已注册图标：** `calendar` / `coin` / `bank` / `warning` / `loanWarning` / `health` / `energy` / `mood` / `reputation` / `map` / `sound` / `vibration` / `stats` / `clear` / `version` / `back` / `reset` / `arrowRight` / `moon` / `sun`

### 7. InteractiveAreaManager — 交互区域管理器

| 项目 | 说明 |
|:---|:---|
| **路径** | [js/components/InteractiveAreaManager.js](js/components/InteractiveAreaManager.js) |
| **导出** | `default class` |
| **依赖** | `IconManager` |
| **用途** | 统一管理 Canvas 场景中的可点击热点区域。支持 hitArea（点击区）和 labelArea（标签区）分离，使用百分比定位自动适配不同屏幕。 |

**关键 API：**

```javascript
const areaManager = new InteractiveAreaManager(game)

// 注册区域
areaManager.register('myArea', {
    label: '区域名',
    action: () => { },
    hitArea: { xPercent: 0.5, yPercent: 0.5, widthPercent: 0.2, heightPercent: 0.15 },
    labelArea: { xPercent: 0.5, yPercent: 0.45, height: 24, paddingX: 10 },
    style: { labelBgColor: 'rgba(255,255,255,0.85)', textColor: '#000', fontSize: 12 }
})

// 触摸事件
areaManager.handleTouchStart(x, y, screenW, screenH)
areaManager.handleTouchEnd(x, y, screenW, screenH)

// 渲染（在场景 render() 中调用）
areaManager.render(renderer)

// 调试模式（可视化区域边框）
areaManager.setDebugMode(true)
```

### 12. SceneTransitionManager — 场景切换过渡动画

| 项目 | 说明 |
|:---|:---|
| **路径** | [js/utils/sceneTransitionManager.js](js/utils/sceneTransitionManager.js) |
| **导出** | `default class` |
| **依赖** | `IconManager` |
| **用途** | 统一管理所有场景切换的过渡动画。支持两种视觉风格：旅途风（前往地点，暖米色遮罩 + 走路小人 + 星火粒子）和展开风（打开地图，深灰蓝遮罩 + 地图图标弹入 + 坐标点闪烁）。自动感知目标场景是否加载完成，三阶段时序（入场→保持→退场），动画期间阻止触摸穿透。 |

**关键 API：**

```javascript
const stam = new SceneTransitionManager(game)
stam.startTravel('sceneWithBackground', '前往 银行...')   // 旅途风
stam.startExpand()                                          // 展开风
stam.update(dt)
stam.render(renderer)
stam.isActive()
```

**两种风格触发规则：**

| 风格 | 触发条件 | 遮罩色 | 动画元素 |
|:---|:---|:---|:---|
| 旅途风 | 所有非 `map` 场景切换 | `#FFF5E6` | 足迹印记 + 星火粒子 + 目的地文字 |
| 展开风 | `switchTo('map')` | `#E0F0FF` | 地图图标弹入 + 坐标点闪烁 + "探索城市..." |

**时序（对应场景切换流程）：**
```
入场 200ms → 保持≥400ms（等待目标场景 onEnter 完成）→ 退场 300ms
```

## 附录：使用模式速查

### 单例模式（直接 import 使用）

| 模块 | 导入语句 |
|:---|:---|
| IconManager | `import iconManager from '../components/IconManager.js'` |
| animationManager | `import animationManager from '../utils/animationManager.js'` |
| imageManager | `import imageManager from '../utils/imageManager.js'` |

### 场景内实例化（继承 Scene 类使用）

| 模块 | 导入语句 |
|:---|:---|
| Renderer | `const renderer = new Renderer(ctx, w, h)` → 由 Game 统一创建 |
| SceneManager | `const sm = new SceneManager(game)` → 由 Game 统一创建 |
| UIManager | `const ui = new UIManager(game)` → 由 Game 统一创建 |
| GameState | `const gs = new GameState()` → 由 Game 统一创建 |
| BackGameManager | `const bgm = new BackGameManager(game)` → 在 HomeScene 中创建 |
| SleepTransitionManager | `const stm = new SleepTransitionManager(game, bedPos, callback)` → 在 HomeScene 中创建 |
| InteractiveAreaManager | `const am = new InteractiveAreaManager(game)` → 在各 Scene 中创建 |
| FloatPhoneButton | `const fb = new FloatPhoneButton(game, opts)` → 按需创建 |
| 各 System 类 | 通常由 BackGameManager 或 Scene 中按需创建 |

### 静态工具（直接调用方法）

| 模块 | 导入语句 |
|:---|:---|
| SVGIcons | `import SVGIcons from '../utils/svgIcons.js'` → `SVGIcons.drawCoin(...)` |
| initCloud | `import { initializeCloudDatabase } from '../utils/initCloud.js'` |
| resetGame | `import { restartGame } from '../utils/resetGame.js'` |
