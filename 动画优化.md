# 🎨 场景过渡动画升级设计方案

## 📊 现状分析

当前 `sceneTransitionManager.js` 的两个动画存在以下不足：

| 动画 | 当前问题 |
|------|---------|
| **旅行动画** | 仅8个圆点+1个箭头+7个随机粒子，没有"旅程"的空间感和方向感 |
| **展开动画** | 仅1个面板缩放+12个随机闪烁坐标点，没有"折叠地图展开"的层次和内容 |

---

## 一、🏃 旅行动画（Travel）升级方案

**核心理念**：「从出发地到目的地的微型旅程」——让玩家感受到真实的"在路上"。

### 视觉分层（5层）

```
┌─────────────────────────────┐
│  L5: 场景标签 + 进度信息      │
│  L4: 行走人物剪影 + 足迹粒子  │
│  L3: 贝塞尔曲线道路 + 路标    │
│  L2: 城市天际线剪影           │
│  L1: 渐变天空背景 + 云朵      │
└─────────────────────────────┘
```

### 各层详细设计

#### L1 — 渐变天空背景
- 用 `ctx.createLinearGradient()` 从顶部浅蓝渐变到底部暖橙（模拟黄昏旅途）
- 3~4朵缓慢漂移的云（`ctx.ellipse()` 组合），速度各不相同
- 几颗呼吸闪烁的星点（按时间正弦波控制 alpha）

#### L2 — 城市天际线剪影
- 屏幕底部绘制一排建筑物剪影（用矩形 + 三角形屋顶拼成）
- 建筑颜色：底部深色半透明 `rgba(45,52,54,0.15)`
- 天际线随道路方向微微偏移，增加视差感

#### L3 — 贝塞尔曲线道路
- 用 `ctx.bezierCurveTo()` 绘制一条从左侧（起点）到右侧（终点）的蜿蜒道路
- 道路主体由两条平行曲线 + 中间虚线组成（模拟地图路线）
- 虚线使用 `setLineDash([8, 12])`，并且虚线偏移量随时间变化产生"流动感"
- 起点放小房子图标 🏠，终点放目标建筑图标 + 脉冲光环
- 道路上散布 6~8 个微型路标（小三角/菱形），alpha 随位置变化

#### L4 — 行走人物剪影 + 足迹粒子
- 一个 Q 版小人剪影（圆头 + 梯形身体 + 两条小腿），沿贝塞尔曲线行进
- 行进使用 **easeInOutCubic** 缓动：起步慢 → 中途快 → 到达慢
- 人物身体带微小弹跳（`Math.sin(time * 8) * 2`），模拟行走
- 身后留下 5~8 个逐渐缩小+淡出的足迹圆点（alpha 从 0.6 → 0）
- 人物到达终点时，触发一个微型"到达绽放"（6个粒子从终点向外弹射）

#### L5 — 场景标签 + 进度信息
- 顶部居中显示「🏙 正在前往 {目的地}...」
- 文字带逐字弹跳出现效果（每个字延迟 50ms，Y 轴偏移正弦波）
- 文字下方一条微型进度条（道路缩影），填充色随人物位置变化

### 时序设计

| 阶段 | 时间 | 效果 |
|------|------|------|
| fadeIn | 0~0.2s | 天空渐变 + 天际线淡入，人物出现在起点 |
| hold | 0.2~0.6s | 人物沿道路行进（0.6s 内完成全程），云朵漂移 |
| fadeOut | 0.6~0.9s | 整体淡出，人物到达终点并触发绽放粒子 |

## 二、🗺️ 展开动画（Expand）升级方案

**核心理念**：「折叠的地图在你面前徐徐展开」——模拟真实纸张展开的触感与仪式感。

### 视觉分层（4层）

```
┌─────────────────────────────┐
│  L4: 地图探索元素（图标弹跳+指南针） │
│  L3: 微型城市地图（网格+区域+建筑） │
│  L2: 折痕线动画               │
│  L1: 纸张纹理遮罩背景          │
└─────────────────────────────┘
```

### 各层详细设计

#### L1 — 纸张纹理遮罩
- 底色用 `#F5ECD7`（仿牛皮纸色）替代当前纯色 `#E0F0FF`
- 背景上绘制微弱的纤维纹理线（3~5条随机角度细线，alpha 0.03）
- 四角添加径向渐变暗角（`ctx.createRadialGradient()` 四角 → 中心，alpha 0→0）

#### L2 — 折痕线动画（核心亮点）
- 面板上有 **3条折痕线**：1条水平 + 2条垂直（模拟折叠地图的折痕）
- 折痕线 = 一条暗线 + 紧邻一条亮线（1px间距），模拟纸张折痕的凹凸感
- **阶段1 (0~30%)**：折痕线从面板中心明显（alpha 0.4）→ 向两侧逐渐消失（alpha 0）
- 面板 scaleY 从 0 展开到 1（模拟纸张从折叠→摊开），同时 scaleX 从 0.3 → 1

#### L3 — 微型城市地图
画的地图面板不再是图标+文字的空壳，而是一幅微型城市地图：

```
┌──────────────────────────┐
│  🏠 住宅区    🏪 商业区   │
│  ┌────┐     ┌──────┐    │
│  │绿块│═════│ 蓝块  │    │
│  └────┘ ║   └──────┘    │
│  ══ 🌊 河流 ══════     │
│  🏭 工业区  📍 你在这里  │
└──────────────────────────┘
```

具体元素：
- **街道网格**：淡灰线（`#D5C8B5`，alpha 0.3），水平3条+垂直3条，从中心向外逐渐绘制
- **区域色块**：2~3个圆角矩形半透明填充
  - 绿色半透明 `rgba(144, 190, 109, 0.25)` = 住宅区
  - 蓝色半透明 `rgba(100, 180, 220, 0.25)` = 商业区
  - 灰色半透明 `rgba(160, 150, 140, 0.25)` = 工业区
- **弯曲河流**：用 `bezierCurveTo()` 画一条蓝色曲线穿过地图
- **当前位置标记**：红色脉冲圆点 + 一圈扩散波纹（alpha 从 0.8 → 0，循环）

#### L4 — 地图探索元素
- 使用 `iconManager` 的图标（home, shop, hospital, bank 等），散布在面板上的对应区域
- 每个图标有 **easeOutBack** 弹跳出现动画（scale 0 → 1.15 → 1，回弹感）
- 图标依次出现，间隔 80ms（从左到右，从上到下）
- 右上角一个微型指南针（圆形 + 菱形指针），缓慢旋转
- 「探索城市...」文字改为优雅衬体线，下方加一条装饰性波浪线

#### 面板展开时序

| 子阶段 | 时间 | 效果 |
|--------|------|------|
| 折叠打开 | 0~0.2s | scaleY: 0→1, scaleX: 0.3→1，折痕线从明显→消失 |
| 骨架绘制 | 0.15~0.35s | 街道网格线依次绘制，区域色块淡入 |
| 内容浮现 | 0.3~0.5s | 图标依次弹跳出现，河流渐显，位置标记脉冲 |
| 稳定呼吸 | 0.5s~ | 面板整体微呼吸（scale ±1.5%），指南针旋转 |

## 三、🛠️ 技术实现路径

### 需要新增的辅助函数

```javascript
// 缓动函数
easeOutBack(t)   // t => 1 + 2.7 * Math.pow(t - 1, 3) + 1.7 * Math.pow(t - 1, 2)
easeInOutCubic(t) // t < 0.5 ? 4*t*t*t : 1 - Math.pow(-2*t+2, 3)/2

// 贝塞尔曲线上的点
pointOnBezier(t, p0, p1, p2, p3) // 三阶贝塞尔

// 路径角度（用于旋转人物朝向）
angleOnBezier(t, p0, p1, p2, p3)
```

### 需要修改的方法

| 方法 | 改什么 |
|------|--------|
| `startTravel()` | 新增贝塞尔曲线控制点、人物状态、足迹队列、地标位置 |
| `startExpand()` | 新增折痕线数据、地图网格数据、区域色块、图标位置列表 |
| `renderTravelStyle()` | 重写为5层渲染管线 |
| `renderExpandStyle()` | 重写为4层渲染管线 |
| `updateAnimElements()` | 新增人物沿路径更新、足迹粒子生命周期、图标弹跳计时 |

### Canvas API 使用清单

| 效果 | API |
|------|-----|
| 渐变天空 | `createLinearGradient()` |
| 暗角效果 | `createRadialGradient()` |
| 贝塞尔道路 | `bezierCurveTo()` |
| 虚线流动 | `setLineDash()` + `lineDashOffset` |
| 人物剪影 | `arc()` + `moveTo()` + `lineTo()` |
| 云朵形状 | `ellipse()` × N 组合 |
| 折痕线 | 相邻两条 line，一暗一亮 |
| 图标弹跳 | `translate` + `scale` + easeOutBack |

## 四、📐 配置化参数

```javascript
this.config = {
  travel: {
    speed: 0.6,           // 人物行进时长(秒)
    footprintCount: 12,   // 足迹粒子数
    cloudCount: 3,        // 云朵数
    showProgress: true,   // 显示进度条
  },
  expand: {
    gridRows: 3,          // 街道网格行数
    gridCols: 3,          // 街道网格列数
    iconBounceInterval: 80, // 图标弹跳间隔(ms)
    creaseCount: 3,       // 折痕线数量
    showCompass: true,    // 显示指南针
  }
}
```

## 五、🎯 预期效果对比

| 维度 | 当前 | 升级后 |
|------|------|--------|
| 旅行空间感 | ⭐ 一条直线+箭头 | ⭐⭐⭐⭐⭐ 蜿蜒道路+人物行进+天际线 |
| 展开仪式感 | ⭐ 一个面板弹出 | ⭐⭐⭐⭐⭐ 折痕展开+城市地图拼出 |
| 视觉层次 | ⭐⭐ 2层 | ⭐⭐⭐⭐⭐ 4~5层 |
| 动效丰富度 | ⭐ 纯缩放+正弦闪 | ⭐⭐⭐⭐⭐ 缓动弹跳+路径动画+粒子 |
| 内容叙事 | ⭐ 图标+文字 | ⭐⭐⭐⭐⭐ 微缩城市+探索引导 |
| 代码可维护性 | ⭐⭐⭐ 平铺直叙 | ⭐⭐⭐⭐ 分层渲染+配置化 |
