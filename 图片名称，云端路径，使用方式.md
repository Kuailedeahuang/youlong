# images集合记录字段设计

## 字段说明

name：图片名称  
fileID：微信云平台调用地址  
note：图片注释  
createTime：创建时间  
type：图片类型,微信云平台云端存储文件夹名称

## 示例

{
  "name": "BaoZhi.png",
  "fileID": "cloud://xxx",
  "note": "这里写中文注释",
  "createTime": "2026-04-13 00:00:00",
  "type": "background"
}
